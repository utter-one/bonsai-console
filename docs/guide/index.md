# What is Bonsai Console?

Bonsai Console is an admin panel for managing the Bonsai conversational AI platform. It provides tools to design AI assistant behavior, monitor live conversations, and administer platform infrastructure.

## Sections

### Design
Configure the behavior of your AI assistants on a per-project basis. Define personas, conversation stages, classifiers, and actions.

### Monitor
Observe conversations in real time, review historical interactions, manage users, and track issues.

### Administration
Manage the platform infrastructure: projects, environments, LLM/TTS providers, API keys, and admin user accounts.
