# Monitor Overview

The **Monitor** section gives you visibility into what's happening in your conversations. Use it to observe live interactions, review historical transcripts, manage end users, and investigate issues.

## What You'll Find

| View | Purpose |
|---|---|
| [Conversations](./conversations) | Browse live and completed conversations with full event details |
| [Users](./users) | View end-user profiles and their conversation history |
| [Issues](./issues) | Review flagged conversations that may need attention |
| [Audit Logs](./audit-logs) | See a record of every change made by admins in the console |

## Project Scope

Monitor views can be filtered by the currently selected project. Select a project from the dropdown in the top navigation bar to see only that project's data, or view across all projects.
