**Providers** are integrations with external AI services such as language model (LLM) APIs and text-to-speech (TTS) services.

## Provider Types

- **LLM** — language model providers (e.g., OpenAI, Anthropic, Azure OpenAI)
- **TTS** — text-to-speech providers for voice output

## Fields

- **Name** — display name for the provider configuration
- **Type** — provider category (LLM or TTS)
- **API Key** — credential used to authenticate with the provider
- **Base URL** — optional override for self-hosted or proxy endpoints
