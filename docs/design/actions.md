**Actions** define automated behaviors that execute during a conversation, such as sending messages, calling APIs, or transitioning to another stage.

## Action Types

- **Message** — send a static or templated message
- **Set Variable** — store a value in conversation context
- **Stage Transition** — move conversation to another stage
- **Tool Call** — invoke an external tool
