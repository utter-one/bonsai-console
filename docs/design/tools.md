**Tools** are callable functions that the AI assistant can invoke to retrieve information or perform operations during a conversation.

## Fields

- **Name** — function name used by the model
- **Description** — explains to the model when and how to use the tool
- **Parameters** — JSON Schema defining the tool's input parameters
- **Endpoint** — backend URL the platform calls when the tool is invoked
