## Prerequisites

- Access credentials (email + password) provided by your platform administrator.
- A compatible modern web browser.

## Logging In

Navigate to the Bonsai Console URL and enter your admin credentials on the login page. On successful authentication you will be directed to the Dashboard.

## Selecting a Project

Most features in the **Design** and **Monitor** sections are scoped to a project. Use the project selector in the top navigation bar to switch between projects.
