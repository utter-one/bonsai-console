## Sidebar

The left sidebar provides navigation between the three main sections:

- **Design** — project-specific AI configuration
- **Monitor** — conversation and user monitoring
- **Administration** — platform management

## Project Selector

The project selector is displayed in the top bar when you are in the Design or Monitor sections. Changing the selected project reloads all data for that project.

## Dark Mode

Toggle between light and dark themes using the icon in the top-right corner of the navigation bar.
