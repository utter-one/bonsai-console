**Classifiers** detect intents and route conversations to different stages or trigger specific actions based on user input.

## Fields

- **Name** — identifier for the classifier
- **Description** — what this classifier detects
- **Labels** — set of classes the classifier can output
- **LLM Settings** — model used for classification
