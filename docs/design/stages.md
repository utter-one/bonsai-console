**Stages** model multi-step conversation flows. Each stage represents a point in the conversation with its own instructions and set of actions.

## Fields

- **Name** — identifier for the stage
- **Description** — purpose of this stage
- **Instructions** — stage-specific instructions appended to the persona's system prompt
- **Actions** — ordered list of actions available in this stage
