A **persona** defines the identity, tone, and core instructions of an AI assistant. Each project can have multiple personas.

## Fields

- **Name** — display name for the persona
- **Description** — internal notes about this persona's purpose
- **Instructions** — system prompt that shapes the assistant's behavior
- **LLM Settings** — language model and inference parameters
- **TTS Settings** — text-to-speech voice and audio configuration
